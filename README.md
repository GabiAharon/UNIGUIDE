# 🛠️ מערכת ניהול תוכן - המדריך המלא ליוניסטרים

מערכת ניהול תוכן מתקדמת עבור המדריך המלא למובילי ומנהלי מרכזי יזמות, הכוללת ממשק עריכה ויזואלי, מערכת בחינות דינמית, וניהול תמונות אוטומטי.

## ✨ תכונות עיקריות

### 📝 ניהול תוכן
- **עריכה ויזואלית** - ממשק ידידותי לעריכת תוכן ללא קוד
- **ניהול קטגוריות** - ארגון אוטומטי של התוכן לפי נושאים
- **קישורים דינמיים** - הוספה וניהול של קישורים רלוונטיים
- **חיפוש מתקדם** - מנוע חיפוש מובנה בתוכן

### 🧪 מערכת בחינות
- **בחינות דינמיות** - יצירה ועריכה קלה של שאלות
- **ערבוב שאלות** - שאלות בסדר אקראי בכל בחינה
- **משוב מיידי** - תוצאות מפורטות עם הסברים
- **סטטיסטיקות** - מעקב אחר ביצועים וציונים

### 🖼️ ניהול תמונות
- **העלאה קלה** - גרירה ושחרור או בחירת קבצים
- **אופטימיזציה אוטומטית** - דחיסה וגדלים מרובים
- **ניהול מתקדם** - ארגון ותיוג תמונות

### ☁️ אינטגרציה עם MCP
- **שמירה בענן** - גיבוי אוטומטי באמצעות Netlify
- **סנכרון אוטומטי** - עדכון התוכן בזמן אמת
- **פרסום אוטומטי** - עדכון האתר בלחיצת כפתור

## 🚀 התקנה והפעלה

### דרישות מערכת
- Node.js גרסה 18 ומעלה
- npm או yarn
- (אופציונלי) חשבון Netlify לפרסום

### התקנה ראשונית

1. **שכפול הפרויקט:**
```bash
git clone https://github.com/unistream/uniguide.git
cd uniguide
```

2. **התקנת תלויות:**
```bash
npm install
```

3. **הרצת הגדרה ראשונית:**
```bash
npm run setup
```

4. **הפעלת שרת פיתוח:**
```bash
npm run dev
```

5. **פתיחת ממשק הניהול:**
```
http://localhost:8888/admin.html
```


## 🎯 שימוש במערכת

### ניהול תוכן

1. **הוספת נושא חדש:**
   - פתח את ממשק הניהול
   - לחץ על "ניהול תוכן"
   - מלא את פרטי הנושא החדש
   - לחץ "הוסף נושא"

2. **עריכת תוכן קיים:**
   - בחר את הפריט מהרשימה
   - לחץ "ערוך"
   - עדכן את התוכן
   - שמור שינויים

3. **ניהול קטגוריות:**
   - בחר קטגוריה מהרשימה הנפתחת
   - התוכן יתארגן אוטומטית לפי הקטגוריה

### ניהול בחינות

1. **הוספת שאלה:**
   - עבור ל"ניהול בחינות"
   - כתב את השאלה והתשובות
   - בחר את התשובה הנכונה
   - שמור

2. **עריכת שאלה קיימת:**
   - בחר שאלה מהרשימה
   - לחץ "ערוך"
   - עדכן ושמור

### ניהול תמונות

1. **העלאת תמונה:**
   - עבור ל"ניהול תמונות"
   - גרור תמונה או לחץ להעלאה
   - הוסף תיאור
   - שמור

2. **שימוש בתמונה:**
   - העתק את הקישור
   - הדבק בתוכן או בקישורים

### פרסום שינויים

1. **גיבוי:**
   - עבור ל"גיבוי ושחזור"
   - לחץ "צור גיבוי"
   - שמור את קובץ הגיבוי

2. **פרסום:**
   - לחץ "פרסם שינויים"
   - המתן לסיום הפרסום
   - בדוק את האתר המעודכן

## ⚙️ הגדרות מתקדמות

### חיבור ל-Netlify

1. **יצירת חשבון Netlify:**
   - הירשם ב-netlify.com
   - חבר את הפרויקט ל-GitHub
   - הגדר את משתני הסביבה

2. **הגדרת Build:**
```toml
# netlify.toml
[build]
  publish = "public"
  command = "npm run build"
```

### משתני סביבה

צור קובץ `.env` עם המשתנים הבאים:
```env
URL=https://your-site.netlify.app
NODE_ENV=production
```

### התאמה אישית

1. **עיצוב:**
   - ערוך את קבצי ה-CSS ב-`admin.html`
   - התאם צבעים וגופנים לפי הצורך

2. **תבניות:**
   - ערוך את התבניות ב-`templates/`
   - הוסף שדות מותאמים אישית

## 🆘 פתרון בעיות

### בעיות שכיחות

1. **השרת לא עולה:**
   - בדוק שיש לך Node.js מותקן
   - הרץ `npm install` שוב
   - בדוק שאין תהליכים שתופסים את הפורט

2. **הנתונים לא נשמרים:**
   - בדוק שיש הרשאות כתיבה לתיקייה
   - וודא שה-localStorage פועל בדפדפן

3. **התמונות לא נטענות:**
   - בדוק את גודל הקבצים (מקסימום 5MB)
   - וודא שהפורמט נתמך (JPG, PNG, WebP, GIF)
