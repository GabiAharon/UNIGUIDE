מה זה העורך החדש?

עורך תוכן מתקדם שמאפשר לך לערוך את index.html ו-quiz.html ישירות בדפדפן, עם שמירה אוטומטית לגיטהאב בזמן אמת!

  • 🔐 הגנת סיסמה מתקדמת
  • 💾 שמירה ישירה לגיטהאב
  • 👀 תצוגה מקדימה בזמן אמת
  • 🎯 כלי עריכה מובנים
  • 📊 סטטיסטיקות ומידע

1הגדרת GitHub Token (פעם ראשונה)

כדי שהעורך יוכל לשמור ישירות לגיטהאב, צריך ליצור Personal Access Token:

צעדי יצירת הטוקן:

  1. עבור ל-GitHub Settings → Personal access tokens
  2. לחץ על "Generate new token (classic)"
  3. תן שם לטוקן: "UNIGuide Editor"
  4. בחר את ההרשאות הבאות:
    • ✅ repo (Full control of private repositories)
    • ✅ public_repo (Access public repositories)
  5. לחץ "Generate token"
  6. חשוב! העתק את הטוקן מיד - הוא יוצג רק פעם אחת
⚠️ אבטחה: הטוקן הזה נותן גישה לגיטהאב שלך. שמור אותו במקום בטוח ואל תשתף אותו עם אף אחד!

2הגדרת העורך

עכשיו נכניס לעורך ונגדיר אותו:

  1. פתח את secure-editor.html
  2. הכנס את הסיסמה: unistream2024
  3. הכנס את הטוקן שיצרת
  4. הכנס את שם הרפוזיטורי שלך בפורמט: USERNAME/REPOSITORY
  5. לחץ "שמור הגדרות"
🎉 מעולה! עכשיו העורך מוכן לעבודה ויוכל לשמור ישירות לגיטהאב!

4תכונות מתקדמות

⚡ שמירה אוטומטית

העורך שומר אוטומטית לגיטהאב כל 30 שניות אם יש שינויים.

🎯 כלי הוספת תוכן

  • הוסף תכנית: מוסיף תבנית לתכנית חדשה
  • הוסף שאלה נפוצה: מוסיף שאלה ותשובה חדשה
  • הוסף קטגוריה: מוסיף כותרת קטגוריה חדשה

⌨️ קיצורי מקלדת

  • Ctrl+S: שמירה מהירה לגיטהאב
  • Enter בכניסה: כניסה מהירה

5טיפים לעריכה

💡 טיפים חשובים:

  • גיבוי: לפני עריכות גדולות, עשה commit ידני בגיטהאב
  • בדיקה: השתמש בתצוגה המקדימה לבדוק שהכל נראה טוב
  • שמירה: לחץ על "שמור לגיטהאב" אחרי שינויים חשובים
  • HTML: שמור על תחביר HTML נכון למניעת שגיאות

🔒 הערות אבטחה חשובות

  • סיסמה: ניתן לשנות את הסיסמה בקוד העורך (החלף את 'unistream2024')
  • טוקן: הטוקן נשמר במחשב שלך בלבד (localStorage)
  • גישה: רק מי שיודע את הסיסמה יכול להשתמש בעורך
  • היסטוריה: כל השינויים נשמרים בהיסטוריית הגיטהאב
