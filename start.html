סנכרון ענן

גיבוי אוטומטי ופרסום באתר באמצעות אינטגרציה עם Netlify

🚨 לפני שמתחילים:

אם זו הפעם הראשונה שלך, צריך להגדיר את החיבור לגיטהאב כדי שהפרסום יעבוד.

⭐ מומלץ: לחץ כאן להגדרה מהירה של גיטהאב

🎯 התחל לעבוד 🔧 הגדרת גיטהאב 📚 צפה במדריך

📋 הוראות הפעלה

  1. הגדרת גיטהאב: לחץ על "הגדרת גיטהאב" להתחברות לגיטהאב (פעם ראשונה בלבד)
  2. פתיחת הממשק: לחץ על "התחל לעבוד" או עבור ל-admin.html
  3. עריכת תוכן: השתמש בממשק לעריכה והוספת תוכן
  4. פרסום מקומי: לחץ "פרסם שינויים מקומית" לעדכון הקבצים
  5. פרסום לאתר: לחץ לחיצה כפולה על publish.bat או הרץ npm run update
  6. תוצאה: האתר שלך מתעדכן אוטומטית ב-Netlify! 🎉

💡 טיפ: למדריך מפורט על השימוש במערכת, צפה ב-README.md
