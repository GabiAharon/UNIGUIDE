📖 הוראות מפורטות: לחץ כאן למדריך המלא

⚛️ עורך React (מומלץ ביותר!) 📖 הוראות שימוש ⚙️ כלי ישן (מקומי) 👀 צפה במדריך

📋 הוראות הפעלה - עורך מאובטח

  1. הגדרת GitHub Token: יצירת טוקן אישי בגיטהאב (פעם ראשונה בלבד)
  2. כניסה לעורך: לחץ על "עורך מאובטח" והכנס סיסמה
  3. הגדרת חיבור: הכנס את הטוכן ושם הרפוזיטורי
  4. עריכת תוכן: ערוך ישירות את index.html או quiz.html
  5. שמירה אוטומטית: השינויים נשמרים אוטומטית לגיטהאב!
  6. תוצאה: האתר שלך מתעדכן בזמן אמת! 🎉

💡 טיפ: למדריך מפורט על השימוש במערכת, צפה ב-README.md או בהוראות השימוש
